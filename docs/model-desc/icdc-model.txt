adverse_event_grade	enum

analysis_area_percentage_tumor	for the NCATS study, percentage of the analysis area represnted by tumor tissue

arm	Where applicable, the nature of each arm into which the study/trial has been divided. For example, in multiple agent clinical trials, the name of the therapeutic agent used in any given study arm
arm_description	Short description of the study arm
arm_id	A unique identifier via which study arms can be differentiated from one another across studies/trials
authorship	A list of authors for the cited work, specifically, for publications with no more than three authors, authorship quoted in full; for publications with more than three authors, authorship abbreviated to first author et al
best_response	indicator as to the best overall response to therapeutic intervention that was observed within an individual patient/subject/donor 
body_surface_area	Called "bsa" in form
body_system	Body system examined; up to 12 in a visit
breed	The specific breed of the canine patient/subject/donor, per the list of breeds officially recognized by the American Kennel Club
canine_individual_id	A unique numerical ID, which, based upon the existence of registration-based matches between two or more study-specific cases, is auto-generated by the data loader, and which thereby tethers matching cases to the single underlying multi-study participant
case_id	The globally unique ID by which any given patient/subject/donor can be unambiguously identified and displayed across studies/trials; specifically the value of patient_id as supplied by the data submitter, prefixed with the appropriate ICDC study code during data transformation
clinical_study_description	A multiple sentence summary of what the study/trial was intended to determine and how it was conducted
clinical_study_designation	A unique, human-friendly, alpha-numeric identifier by which the study/trial will identified within the UI
clinical_study_id	Where applicable, the ID for the study/trial as generated by the source database
clinical_study_name	A succinct, narrative title for the study/trial, exactly as it should be displayed within the UI
clinical_study_type	An arbitrary designation of the study/trial to indicate its underlying nature, e.g. Clinical Trial, Transcriptomics, Genomics
cohort_description	Where applicable, the nature of each cohort into which the study/trial has been divided, e.g. in studies examining the effects of multiple doses of a therapeutic agent, the name and dose of the therapeutic agent used in any given cohort
cohort_dose	The intended or protocol dose of the therapeutic agnet used in any given cohort
cohort_id	A unique identifier via which cohorts can be differentiated from one another across studies/trials
collection_access	Indicator as to whether the image collection can be accessed via download versus being accessible only via the cloud
comment	generic comment
concurrent_disease	Indicator as to whether the patient is has any significant secondary disease condition(s)
concurrent_disease_type	specifics of any notable secondary disease condition(s) within the patient
contact_type	need vocab

date_of_informed_consent	The date upon which the owner of the patient/subject/donor signed an informed consent on behalf of the subject

document_number	S/N of the executed CRF
dose_limiting_toxicity	bool

file_description	optional description of the file and/or its content, e.g. amended versus original pathology report, low resolution versus high resolution image file, etc.

file_name	name of the file, as provided by the data owner
file_size	size of the file as derived by the loader
file_status	enumerated representation of the status of files

program_name	The name of the program under which related studies will be grouped, in full text and unabbreviated form, exactly as it will be displayed within the UI
program_short_description	An abbreviated, single sentence description of the program
program_sort_order	An arbitrarily-assigned value used to dictate the order in which programs are displayed within the UI
publication_title	The full title of the publication stated exactly as it appears on the published work
pubmed_id	Where applicable, the unique numerical identifier assigned to the cited work by PubMed, by which it can be linked to via the internet
registering_institution	TBD
registration_id	Any ID used by a data submitter to identify a patient/subject/donor, either locally or globally
registration_origin	The entity with which each registration ID is directly associated, for example, an ICDC study, as denoted by the appropriate study code, or the biobank or tissue repository from which samples for a study/trial participant were acquired, as denoted by the appropriate acronym
repository_name	The name of the image repository within which the image collection can be found, stated in the form of the appropriate acronym
sample_chronology	indicator as to when a sample was acquired relative to therapeutic treatment administration and/or key disease outcome observations
sample_id	globally unqiue ID for a sample, generated from values within the raw data concatenated together to ensure global uniqueness
sample_preservation	the method by which a sample was preserved
sample_site	the anatomical location from which a sample was acquired
sex	The biological sex of the patient/subject/donor
site_short_name	The widely-accepted acronym for the institution at which the patient/subject/donor was enrolled into the study/trial, and then treated under the appropriate veterinary medicine program
specific_sample_pathology	indicator of the specific pathology associated with a sample, e.g. squamous cell carcinoma
study_disposition	An arbitrarily-assigned value used to dictate how the study/trial is displayed via the ICDC Production environment, based upon the degree to which the data has been on-boarded and/or whether the data is subject to any temporary embargo which prevents its public rlease
summarized_sample_type	summarized representation of a sample's physical nature, normality, and derivation from a primary versus a metastatic tumor, based upon the combination of values in three independent properties
surgical_finding	Findings_FUL on form
total_tissue_area	for the NCATS study, area within the slide's analysis area represented by tissue
treatment_data	Indicator as to whether treatment data for the patient exists
treatment_since_last_contact	y/n
tumor_grade	grade of the tumor from which a sample was acquired as determined by the evaluation of a pathologist
tumor_sample_origin	indicator as to whether a tumor sample was derived from a primary versus a metastatic tumor
tumor_tissue_area	for the NCATS study, area within analyzed tissue area represented by tumor tissue
unexpected_adverse_event	bool
veterinary_medical_center	The full name of the insitution at which the patient/subject/donor was enrolled into the study/trial, and then treated under the appropriate veterinary medicine program, together with the site's city and state
volume_of_tumor	volume of the tumor from which the sample was derived
weight	The weight of the patient/subject/donor at the time of study/trial enrollment and/or biospecimens being acquired, at least in the case of studies that are not longitudinal in nature
width_of_tumor	width of the tumor from which a tumor sample was derived
year_of_publication	The year in which the cited work was published
